# Ironhack Data Analytics Bootcamp

This repo contains all of the practical exercises I did during the Data Analytics Bootcamp @ [Ironhack]( https://www.ironhack.com/en) in Mexico City. The entire part time + 400 hours course lasted for 6 months (Sept 2020 - March 2021). It was divided into 3 modules:

Module 1 - Data Extraction & Transformation;

Module 2 - Data Analysis & Visualization;

Module 3 - Data Modeling.
